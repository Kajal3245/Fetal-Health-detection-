# 🩺 AI Fetal Health Prediction System  

🚀 Live App:  
https://fetal-health-ai-kajal-final.streamlit.app/  

---

## 📌 About the Project  
This project uses Machine Learning to predict fetal health based on clinical parameters.  
It classifies the condition into:
- Normal  
- Suspect  
- Pathological  

---

## ⚙️ Features  
- Simple Streamlit web app  
- Real-time prediction  
- Random Forest + SVM models  
- Performance visualization  

---

## 🧠 Tech Stack  
- Python  
- Streamlit  
- Scikit-learn  
- Pandas, NumPy  

---

## 📊 How to Use  
1. Open the app link  
2. Enter values  
3. Click Predict  
4. See result
